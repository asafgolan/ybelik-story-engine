ybelik story engine · CodePens

an ink-wash view-transition story engine — four packages, live on npm + unpkg

generate → trace → compile → reveal (intent) (auto) (buckets) (paints)

the money pen

scene-compiler + reveal-engine — live compile: scrub direction/jitter/seed, watch the SVG re-bucket in-browser. The linter surfaces on hostile input.

bucket reveal

reveal-engine — scrub a pre-compiled scene bucket-by-bucket with setLevel.

scene player

scene-player — descriptor-driven scenes with captions; forward paints, backward jumps to painted.

the full shell

navigation-shell — scroll to engage, wheel/touch/key gestures, progress dots. The whole experience.
pipeline: generate → trace → compile → reveal · source: github.com/asafgolan/ybelik-story-engine
